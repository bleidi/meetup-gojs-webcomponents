What is?

Are a set of web platform APIs that allow you to create new custom, reusable, encapsulated HTML tags to use in web pages and web apps.

  • Custom elements
  • Shadow DOM
  • HTML template
  • HTML imports

						customElements.define('welcome-aboard', WelcomeAboard);

Shadow DOM

					
						var shadow = this.attachShadow({mode: 'open'});
						var el = document.createElement(...);

						shadow.appendChild(el);


  • close/open-mode
  • encapsulated dom-tree & style
  • "slotable"
